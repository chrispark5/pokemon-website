This app is built with Next.js, React, and Mantine. It allows users to explore all Pokémon, view detailed information about each one, and interact with a type strength and weakness tool to understand Pokémon battle mechanics better.

# Features

• Pokémon List: Displays a list of all Pokémon with key details (name, type, stats, etc.).
• Interactive Type Tool: Allows users to explore the strengths and weaknesses of Pokémon types in battles.
• Detailed Pokémon Information: Provides detailed information about each Pokémon, including stats, evolutions, abilities, and moves.

# Installation

```javascript
//1. Clone repo
git clone {repo_url}
//2. Navigate to repo
cd pokemon-website
// 3. Install dependencies
npm install
// 4. run dev server
npm run dev
//5. open app in browser
Go to http://localhost:3000

```
